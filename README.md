# WebAppEggShop
Project completed for a class. It is a web application that allows for users to create an account, sell eggs, make offers for eggs, and perform other features. 

To run the application go to the project directory and use "npm i" to install dependencies. Then, go the the app.js file and on line 16 set the const MongoUri to be equal to a connection string for a mongodb database. Finally, use "node app" or "nodemon app" to run.
